TUTORIAL DE ACTUALIZACIÓN DE WINE APLICABLE A:
- Ubuntu 18.04 y todos sus derivados oficiales: Lubuntu, Ubuntu Budgie, Ubuntu Kylin, Ubuntu MATE, Ubuntu Studio, Xubuntu y Linux Mint 19.x


SOLUCIÓN INSTALAR WINE 4

1.- Si tu sistema es de 64 bits activa la arquitectura de 32 bits (si no la tienes ya activada)

sudo dpkg --add-architecture i386 

2.- Descarga y añade la llave del repositorio, ponga estas dos líneas de código una por una ejecutandolas

wget -nc https://dl.winehq.org/wine-builds/winehq.key
sudo apt-key add winehq.key



3.- Añade el repositorio para Ubuntu 18.04 o Linux Mint 19.x

sudo apt-add-repository 'deb https://dl.winehq.org/wine-builds/ubuntu/ bionic main'

4.- Luego actualiza todos los repositorios:

sudo apt update

5.- Instale Wine la rama estable:

sudo apt install --install-recommends winehq-stable

y una vez actualizado ya aparece la opción de abrir los ejecutables de windows con clic derecho (no ha sido necesario reiniciar):
